## Hello DB

数据库是一个数据集合

* 长期保存在计算机内
* 按照一定数据模型组织
* 被多个用户共享

一个数据库可以有多个表/关系。

一个表

* 有一个主键
* 有很多个字段
* 有很多条记录/元组，一条记录上有很多个值

DBMS 是应用程序和 OS 之间的软件，如

* Acccess
* MySQL
* SQL Server
